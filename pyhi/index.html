pyhi: Python Package Structure and Distribution Demo

by Bob Carpenter, Alias-i

pyhi is a bare bones Python package implementing a standard hello world-type program in the context of a ready-to-distribute package. It illustrates

  • package and module structure,
  • configuration for build and installation,
  • running scripts, and
  • unit testing.

Download

pyhi 1.0 is available as a gzipped tarball or zip archive:

  • pyhi-1.0.tar.gz
  • pyhi-1.0.zip

Installation and Use

Download and unpack the distribution, then consult the top-level README.txt file.
