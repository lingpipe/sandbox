Classes for manipulating data from the OMIM, Online Mendelian Inheritance in Man Database.

This database is maintained and distributed as part of the NCBI Entrez search engine. It is available for FTP download from ftp://ftp.ncbi.nih.gov/repository/OMIM/omim.txt.Z See the OMIM FAQ for more information.

The file omim.txt is an ASCII text file containing a set of records. Each record contains a number of fields. Records begin with the header: *RECORD* on its own line. Fields begin with the header: *FIELD* <FIELD KEY> also on its own line. The class {@link com.aliasi.lingmed.omim.OmimParser} parses this file into its constituent {@link com.aliasi.lingmed.omim.OmimRecord}s.
