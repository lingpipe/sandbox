Classes for manipulating data from the NCBI Entrez Gene database. The classes in this package are able to read the XML formatted version of the database distribution. This database is available for FTP download from ftp.ncbi.nih.gov/gene/. Gene data is distributed in compressed binary ASN.1 format, together with a set of tools used to translate the data to XML. The ASN to XML converters are available for FTP download from ftp.ncbi.nlm.nih.gov/asn1-converters/by_program/gene2xml/. Documentation on the data values is found at www.ncbi.nlm.nih.gov/IEB/ToolBox/CPP_DOC/lxr/source/src/objects/entrezgene/entrezgene.asn.

Example of running the converter over an .ags file:

./linux.gene2xml -b <All_Data.ags | grep -v 'NCBI_Entrezgene.dtd' > allData.xml

The output of gene2xml is piped through grep because the EntrezGene DTDs don't validate properly.
